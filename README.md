## Full Description

This is a private space to check in, reflect, and write — without accounts, ads, or tracking.

Men’s Mental Health Journal is designed to be simple, secure, and pressure‑free. Everything stays on your device, protected with encryption and optional biometric or PIN lock.

Whether you’re carrying stress, working through thoughts, or just want a quiet place to write, this journal meets you where you are.

**Features**
• Daily journal entries
• Guided prompts for reflection
• Encrypted local storage
• Optional fingerprint / face / PIN lock
• Daily reminder notifications (opt‑in)
• Offline‑first — no internet required
• No accounts, no ads, no cloud sync

This app does not provide medical advice or diagnosis. It is a personal journaling tool meant to support reflection and self‑awareness.
